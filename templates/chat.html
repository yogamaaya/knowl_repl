Talk to Knowl

Knowl is any Knowledge, personified.

Update the source of knowledge body here and start a conversation!

Default text: Summary of Bhagawad Gita

✨ Knowl is intented to be your fun partner to understand a text from a different perspective of your own.✨
If you choose to change the base text corpus, please be advised that the old data given to Knowl so far from everyone is still remembered. Knowing that, play around as you wish! The way to helpful responses lie in asking helpful questions. Here are some tips to ask effectively:

  • Try prefacing questions with "In this context", "According to this text" etc.!

  • Whenever you paste new text, please ask new and specific questions to get new answers.

  • You can try rephrasing the same question or command if a response doesn't please you!

  • Try to use keywords of the text, and reference the current context as much as possible.

  • General summary style questions are not recommened due to the text base being variable, as the Answering Model retains all information.

    • PS: Please give Knowl some time to think as sometimes responses might be slowwww~
